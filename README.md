# worker-mailer site

Website for `@workermailer/smtp`, built with SvelteKit, Flowbite Svelte,
Tailwind CSS, Bun, and Cloudflare Workers.

## Local development

```bash
bun install
bun run dev
```

## Checks

```bash
bun run check
bun run build
```

## Deploy

```bash
bun run deploy
```

## Routes

- `/` landing page
- `/docs` documentation
- `/examples` copy-ready snippets
- `/changelog` release timeline
- `/emdash` EmDash discussion summary
